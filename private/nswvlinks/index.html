      Wiring guide for connecting Heraeus IM2 SensorLab to the Receiver Box using CAT5/6 and DB9–RJ45 adapters: clarifies

    • Disable QUBE Wireless Safety on SLS:

      Disable QUBE Wireless Safety on SLS.pdf
    • Set SLS To Display Time Interval Between Measurements:
      Directions for setting up the SLS to display the time interval between samples on the screen.
      Quick steps to show the time interval between measurements on SLS:
      log in (password 2448), open User Settings → Display settings → Edit, check “Display interval between measurements,” and Save.
      Note: this isn’t stored in the Settings File backup.

      SLS Celox Bath Level Licensing Procedure.pdf
    • SLS Remote Viewer Installation:
      Directions for installing the SLS Remote Viewer.
      Remote Viewer setup for IM2/SensorLab Steel: download and unzip SensorLab_v3.3.0, run DeployAgent.exe,
      apply the henmachine.reg license key, then connect (scan or by IP) over LAN—supports up to 4 remote PCs on Windows 7+.

    ### Hydris PLC Integration (AOIs / Siemens FCs / Telegrams)

    • Hydris_Siemens_TIA_Portal_Integration_v1:
      Step-by-step integration guide for Siemens TIA Portal (v18) using the FC_HEN_HydrisParser_Hyd2PLC parser.
      Covers source import, UDT generation, overlay tag mapping, Data Block structure, ladder call implementation, and live monitoring of parsed and derived values.
      Includes configuration notes for byte-aligned telegrams, byte order requirements, and correct input address mapping.
      Hydris Siemens Tia Portal Integration v1.pdf
    • HEN_HydrisParser_Hyd2PLC_Siemens_FC:
      Siemens TIA Portal SCL function (FC) and supporting UDTs for parsing Hydris/HydroVAS data using PLC-aligned output telegrams.
      Designed for direct overlay of the Hydris input image into a structured UDT with clean parsed values and derived BOOLs for automation logic.
      Includes FC_HEN_HydrisParser_Hyd2PLC.scl and associated UDT definitions for Parsed and Derived result structures.
      HEN_Hydris_Siemens_FC_v1_0.zip
    • Hydris_Industrial_Output_Telegrams_PLCAligned_v1_High_Byte_and_Low_Byte:
      Hydris industrial output telegram package containing both PLC-aligned variants of the “Industrial Output Telegram PLCAligned v1” definitions:
      High Byte v1 and Low Byte v1. These Hydris→PLC telegrams preserve the same field structure and byte offsets while providing either big-endian or
      little-endian multibyte formatting, reordered for clean PLC-side 4-byte alignment and easier direct overlay into controller data structures with minimal parsing workarounds.
      Hydris Industrial Output Telegrams PLCAligned v1 - High Byte and Low Byte - 04022026.zip
      GM_Hydris_PLC_Aligned_Output_Telegrams_Reference:
      Reference guide describing the Hydris PLC-aligned industrial output telegrams and how they differ from the default Hydris output telegram documented in the manual.
      Includes the purpose of the new High Byte and Low Byte variants, the rationale for the PLC-aligned layout, a comparison to default Table 26 offsets, and the equivalent
      offset table for the new telegram definitions.
      GM_Hydris_PLC_Aligned_Output_Telegrams_Reference - 04022026.pdf
